# core go mod

Module management.

## Subcommands

| Command | Description |
|---------|-------------|
| `tidy` | Tidy go.mod |
| `download` | Download dependencies |
| `verify` | Verify dependencies |
| `graph` | Show dependency graph |

## Examples

```bash
core go mod tidy
core go mod download
core go mod verify
core go mod graph
```
